# Kanye Quotes Web App

## Project Type
Solo project

## Description
A web application that displays random inspirational quotes from Kanye West using the kanye.rest API. Users can fetch new quotes, copy them, share, and view recent and suggested quotes. The project features a modern, responsive UI with theme toggling.

---

## Project Overview
This project is a solo student submission for ELEC3. It demonstrates API integration, dynamic DOM manipulation, and responsive web design. The app fetches and displays random Kanye West quotes, allowing users to interact with the content in various ways.

---

## Main Features
- Fetch random Kanye West quotes
- Copy quotes to clipboard
- Share quotes (if supported)
- View recent and suggested quotes
- Theme toggle (light/dark mode)
- Responsive, modern UI

---

## APIs Used
- **API Name:** kanye.rest
- **Base URL:** `https://api.kanye.rest/`
- **Endpoints:**
  - `/` (GET random quote)
- **Parameters:** None required
- **Authentication:** None required

- CSS3 (with custom gradients and responsive design)
